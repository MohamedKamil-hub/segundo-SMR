# tema 2
un gemelo digital permite tener una copia virtual de una maquina real para saber como esta y controlarla sin tocarla.
**Ejemplos:**
• En una fábrica: los sensores de una máquina envían datos a un panel digital para detectar si hay fallos.
• En un coche: se envía la temperatura del motor y otros datos al ordenador del taller.
• En una casa inteligente: los sensores de luz o temperatura avisan alsistema, y desde el móvil puedes encender las luces o ajustar la calefacción.

cloud computing es un modelo  que permite utilizar recursos informaticos a traves de internet de forma flexible escalable y pagando solo por lo que se usa sin tener infraestructura propia
elasticidad, escalabilidad, alta disponibilidad, pago por uso, acceso global, seguridad avanzada.

**modelos de servicio**
**iaas**
	alquilas infraestructura virtual basica, servidores, almacenamiento y redes
	tu instalas y gestionas el sistema operativo, las aplicaciones y la seguridad interna.
	ec2 por ejemplo es como alquilar un terreno vacio donde tu contruyes lo que quieres

**paas**
	plataforma como servicio, te ofrecen una plataforma lista para desarrollar y desplegar aplicaciones , con sistema operativo, servidor web y herramientas integradas
	==firebase y vercel== por ejemplo es como alquilar un taller con herramientas necesarias
	

**saas**
	software completamente listo para usar desde navegador o app, sin instalacion o mantenimiento
	gmail, canva por ejemplo es como alquilar un coche listo para conducir 

**nube**
**publica**
	infraestructura compartida entre varios clientes de manera aislada y segura, util para startups, aps web o proyectos de escalado rapido

**privada** 
	infraestructura exclusiva para una organizacion con control total, util para bancos gobiernos hospitales y datos sensibles

**hibrida**
	combina nube publica y privada segun necesidades de seguridad y rendimiento
	para empresas grandes o corporaciones con datos criticos



**seguridad** modelo de responsabilidad compartida
- proveedor seguridad fisica, hardware y red
- cliente acceso permisos contraseñas y configuraciones
- medidas esenciales mfa cifrado, iam backups auditorias
