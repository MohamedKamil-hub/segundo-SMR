etquetas basicas

En HTML, las etiquetas básicas sirven para estructurar una página web.

<html>: Contiene todo el código de la página.
<head>: Parte donde van los datos de la página, como título o enlaces a estilos.
<title>: Define el título que aparece en la pestaña del navegador.
<body>: Contiene todo lo visible de la página.
<h1> a <h6>: Encabezados, de más importante a menos importante.
<p>: Párrafos de texto.
<a>: Enlaces a otras páginas o secciones.
<img>: Imágenes.
<ul> y <ol>: Listas, sin orden (ul) o numeradas (ol).
<li>: Elementos dentro de listas.
<div> y <span>: Contenedores para agrupar contenido y aplicar esti

listas basicas

<ul>: Lista sin orden, con viñetas.
<ol>: Lista ordenada, con números.
<li>: Cada elemento de la lista, dentro de <ul> o <ol>.

formularios

<form>: Sirve para crear un formulario donde los usuarios pueden enviar información.
<input>: Campo donde el usuario escribe algo (texto, contraseña, número, etc.).
<textarea>: Área más grande para escribir textos largos.
<select>: Lista desplegable para elegir opciones.
<option>: Cada opción dentro de un <select>.
<button>: Botón para enviar o realizar alguna acción.
