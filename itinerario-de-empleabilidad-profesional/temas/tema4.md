### **Unidad 4: Identificación y Validación de Ideas Emprendedoras**

#### **Sesión 9: Identificación de Problemas y Oportunidades**
- El emprendimiento comienza con la **detección de un problema real** que afecta a un grupo de personas.
- **Mirada emprendedora**: observar el entorno para identificar frustraciones y necesidades no resueltas.
- **De la queja a la oportunidad**: reformular problemas como oportunidades de solución innovadora.
- **ODS (Objetivos de Desarrollo Sostenible)**: marco de Naciones Unidas que sirve como guía para identificar problemas globales y oportunidades de impacto.
- **Tendencias**: cambios sociales, tecnológicos y económicos que indican nuevas oportunidades de mercado.

#### **Sesión 10: Validación Ágil**
- **Validación ágil**: proceso para comprobar hipótesis antes de invertir recursos.
- **Metodología Lean Startup**: ciclo **Construir – Medir – Aprender**.
- **Proceso de validación**:
  1. **Formular hipótesis**: supuestos sobre cliente, problema y solución.
  2. **Entrevistas de problema**: validar que el problema existe y es relevante.
  3. **Diseño de experimentos**: prototipos, landing pages, encuestas, etc.
  4. **Medición y aprendizaje**: decidir si perseverar o pivotar.
- **Principio clave**: hablar con clientes reales antes de desarrollar el producto.

#### **Sesión 11: Herramientas Ágiles**
- **Lean Canvas**: herramienta de una página para plasmar hipótesis del modelo de negocio.
  - Bloques: Problema, Segmento de clientes, Propuesta de valor única, Solución, Canales, Fuentes de ingreso, Estructura de costes, Métricas clave, Ventaja injusta.
- **Customer Development**: proceso para validar hipótesis con clientes reales.
  - Etapas: Descubrimiento, Creación, Validación, Consolidación.
- **Prototipado rápido**: crear versiones simples del producto para testear y aprender.
- **Ciclo de aprendizaje ágil**:  
  **Lean Canvas** → **Customer Development** → **Prototipado rápido** → **Feedback** → **Ajuste**

### **Mitos y Realidades Destacados**
- **Mito**: "Las mejores ideas no tienen competencia."  
  **Realidad**: La competencia valida que existe un mercado.
- **Mito**: "No puedo hablar con clientes hasta tener el producto terminado."  
  **Realidad**: La validación debe hacerse antes de desarrollar.
- **Mito**: "El Lean Canvas es un plan de negocio terminado."  
  **Realidad**: Es un documento vivo que se actualiza con el aprendizaje.
- **Mito**: "Validar es una pérdida de tiempo."  
  **Realidad**: Reduce el riesgo y evita construir algo que nadie quiere.
