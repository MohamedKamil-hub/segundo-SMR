Cheatsheet de Formularios HTML

Guía visual para aprender y recordar elementos de formulario

Estructura Básica

Todo formulario comienza con la etiqueta <form> y puede contener múltiples campos agrupados con <fieldset>.

Ejemplo:

<form>
<fieldset>
<legend>Título del grupo</legend>
<!-- campos aquí -->
</fieldset>
</form>
Tip: Usa <fieldset> para agrupar campos relacionados y mejorar la accesibilidad.

Campos de Texto

Para capturar texto libre del usuario.

<label for="nombre">Nombre:</label>
<input type="text" id="nombre" placeholder="Escribe tu nombre">
Tip: Siempre usa el atributo for en las etiquetas label que coincida con el id del input para mejor accesibilidad.

Botones de Radio

Para seleccionar una única opción entre varias.



<input type="radio" id="amd" name="cpu" value="amd">
<label for="amd">AMD</label>

<input type="radio" id="intel5" name="cpu" value="intel5">
<label for="intel5">Intel i5</label>
Tip: Todos los botones de radio del mismo grupo deben tener el mismo valor en el atributo name.

Casillas de Verificación

Para seleccionar múltiples opciones.



<input type="checkbox" id="matematicas" name="materias" value="matematicas">
<label for="matematicas">Matemáticas</label>

<input type="checkbox" id="historia" name="materias" value="historia">
<label for="historia">Historia</label>
Tip: Usa el atributo checked para que una casilla aparezca seleccionada por defecto.

Listas Desplegables

Para seleccionar una opción de una lista.

<label for="idioma">Idioma:</label>
<select id="idioma" name="idioma">
<option value="español">Español</option>
<option value="ingles">Inglés</option>
<option value="frances">Francés</option>
</select>
Tip: Puedes usar <optgroup> para agrupar opciones dentro de un select.

Área de Texto

Para capturar texto más extenso.


<label for="comentario">Comentario:</label>
<textarea id="comentario" name="comentario" rows="4" cols="40" placeholder="Escribe tu comentario aquí"></textarea>
Tip: Usa los atributos rows y cols para definir el tamaño del área de texto.

Fecha y Hora

Para seleccionar fechas y horas específicas.



<label for="fecha">Fecha:</label>
<input type="date" id="fecha" name="fecha">

<label for="hora">Hora:</label>
<input type="time" id="hora" name="hora">
Tip: También existen datetime-local para fecha y hora juntas, y month y week para seleccionar meses o semanas completas.

Selector de Color

Para seleccionar un color.

<label for="color">Color favorito:</label>
<input type="color" id="color" name="color" value="#3498db">
Tip: Puedes establecer un color por defecto usando el atributo value con un código hexadecimal.

Subida de Archivos

Para permitir al usuario subir archivos.

<label for="archivo">Selecciona un archivo:</label>
<input type="file" id="archivo" name="archivo">
Tip: Para permitir múltiples archivos, añade el atributo multiple al input.
