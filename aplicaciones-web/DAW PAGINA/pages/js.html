introduccion JavaScript

Cheatsheet rápido de JavaScript:
- Seleccionar elementos: document.getElementById("id"), document.querySelector(".clase")
- Cambiar texto: element.textContent = "Nuevo texto"
- Cambiar estilo: element.style.color = "red"
- Escuchar eventos: element.addEventListener("click", function(){ ... })
- Mostrar alerta: alert("Mensaje")
- Ejecutar código al cargar la página: window.onload = function(){ ... }
- Log en consola: console.log("Mensaje")
Con esto puedes hacer interacciones básicas: botones que cambian texto, colores, mostrar mensajes, etc.

ejemplos rápidos de uso:

- Cambiar el texto de un párrafo al hacer clic en un botón:
document.getElementById("miBoton").addEventListener("click", function(){ document.getElementById("miParrafo").textContent = "Texto nuevo"; });
- Cambiar color de fondo de un div al pasar el ratón:
document.getElementById("miDiv").addEventListener("mouseover", function(){ this.style.backgroundColor = "yellow"; });
- Mostrar un mensaje cuando se carga la página:
window.onload = function(){ alert("Página lista"); }
- Escribir en la consola para depurar:
console.log("Esto funciona");
Con estos ejemplos puedes hacer que tu página reaccione al usuario y modificar contenido, estilos o mostrar alertas sin recargarla.
