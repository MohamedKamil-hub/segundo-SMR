DISEÑO DE UNA LANDING PAGE BÁSICA
Módulo: Aplicaciones Web
Trabajo individual
Entrega
Carpeta comprimida .zip o repositorio con:
• index.html
• style.css
• Carpeta img/ (si se usan imágenes)

OBJETIVO DE LA PRÁCTICA
El objetivo de esta práctica es crear una landing page básica, aplicando
correctamente:
• Estructura HTML5
• Estilos CSS sencillos
• Diseño claro y ordenado
• Buenas prácticas mínimas de maquetación

CONTEXTO DE LA PRÁCTICA
Una empresa ficticia necesita una landing page sencilla para presentar un servicio 

 tiene que ser sobre este tema:
• Servicio de mantenimiento de redes

**ESTRUCTURA OBLIGATORIA DE LA WEB**
**La landing page DEBE CONTENER COMO MÍNIMO las siguientes secciones:**

Cabecera (header)
• Nombre de la empresa o servicio
• Un pequeño eslogan o frase descriptiva
Sección principal (main)
Debe incluir:
• Descripción del servicio (mínimo 1 párrafo)
• Lista de servicios (mínimo 3, usando <ul> o <ol>)

4
Sección destacada
• Un bloque visual (div o section) con:
o Un título
o Texto breve
o Fondo de color distinto o borde

Formulario de contacto
Debe incluir:
• Nombre
• Email
• Mensaje
• Botón de enviar
El formulario no tiene que funcionar, solo maquetarse
Pie de página (footer)
• Nombre del autor
• Curso y módulo
• Año actual

5
REQUISITOS DE CSS (OBLIGATORIOS)
El archivo style.css debe incluir:
• Uso de:
o Colores
o Márgenes (margin)
o Rellenos (padding)
o Tipografía (font-family)
• Maquetación básica:
o Centrar contenido
o Ancho máximo (max-width)
• Estilos visibles para:
o Header
o Botones
o Footer

No se permite CSS dentro del HTML
No se permite Bootstrap ni frameworks

6
REQUISITOS TÉCNICOS
✔ HTML5 bien estructurado
✔ Etiquetas semánticas (header, main, section, footer)
✔ CSS externo enlazado correctamente
✔ Código limpio y ordenado
✔ Nombres de archivos correctos
