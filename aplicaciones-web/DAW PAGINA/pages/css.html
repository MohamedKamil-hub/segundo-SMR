<style>: Aquí escribes los estilos CSS dentro de la página HTML.

color: Cambia el color del texto.
background-color: Cambia el color de fondo.
font-size: Cambia el tamaño de la letra.

margin: Espacio afuera del elemento.
padding: Espacio dentro del elemento.
